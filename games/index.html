Apr 2018
Porsche Ring - my first 3D Godot game.
It's a racing game, for which I've created everything myself, except Godot-boy model and track texture. I like that Porsche model, but it needs more work. I don't have a page for it in Blender section yet, but if you like it and/or need it, I can fix remaining issues and provide a download. The game features standard Godot car physics, it has a lap with checkpoints and lap timer. Actually fun to play. Try to get sub 20 sec. You can see an attempt of making a modular track on the side, but it was abandoned, as Godot didn't (doesn't still?) have hex-tileset workflow, and I didn't want to make is square-based.
It is not optimised at all, so you can try to play Porsche Ring online, if your PC can handle it, or download a Windows build of it, it will run just fine.

Dec 2017
Tic Tac Toe - my first "big" Godot game.
It is a 3x3 and 5x5 Tic Tac Toe game, where you can play hotseat with friend, 3x3 TTT against "simple AI" (it just fills random empty spots on it's turn), or launch 2 builds in local network and establish a chat between 2 computers. Unfortunately, you will have to manually make your opponent's moves on your field :)
It was supposed to have full multiplayer and a strong minimax AI, but I had no idea how to create them at the time. This is still certainly a non-featureless project in other spheres, and that's why I post it.
You can try it online, but you should definitely check out the Windows build and play with it's "multiplayer" stuff.
